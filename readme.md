# Note App

A browser based, offline first note taking app. It should be like Notes app in OSX when finished.

### Stack
React, React router, Babel, Webpack, Material-UI, Netlify

### Done
* Project bootstrapped
* Build and deploy
* UI layout
* Redux integration
